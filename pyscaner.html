2. Python API

API

Scaner API class

SensorWaitMode

Defines the behaviour of EasyScaner while it waits for the sensors to fully load

Vehicle

Vehicle class, describing vehicle instance inside SCANeR simulation

VType

Vehicle types

CruiseControlMode

Describes the currently active mode of cruise control

Indicator

Describes what indicators are on.

LightsInfo

Describes the status of all the external lights on a vehicle

NumericalCommand

Describes a command that has been given to a vehicle, where the command is based on a float.
