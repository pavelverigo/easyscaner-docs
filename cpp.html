struct numerical_command
#include <Vehicle.hpp>

Describes a command that has been given to a vehicle, where the command is based on a float.

Examples include orders to move or accelerate at a certain speed, using Vehicle::set_speed() or Vehicle::set_accel(). You can obtain an command that has been given to a vehicle with functions such as Vehicle::get_accel_command().

enum sensor_wait_mode

Defines the behaviour of EasyScaner while it waits for the sensors to fully load.

Values:

enumerator NO_WAIT

Default behaviour, do not wait for the sensors to start sending data.

enumerator DELAY_RUN

EasyScaner does not come into effect until the sensors start sending data, autonomous vehicles may still be controlled by the TRAFFIC process until then.

enumerator FREEZE_VEHICLES

All vehicles are forced to stand still until the sensors start sending data.
